## 8:00 System Startup Checklist

Complete this checklist when arriving first (around 8am) to prepare the system before other team members arrive. This is typically done by the first Director or Producer on site.

### In The Room

- [ ] Switch system on (if not already on)
- [ ] Switch lighting desk on
- [ ] Switch on cameras
- [ ] Switch on host light
- [ ] Switch on Camera 6
- [ ] Get out Camera 1 and Camera 3

### Upstairs in Gallery

- [ ] Hit system initial setup button (push and hold style)
- [ ] Check stream audio feed in Allen & Heath Custom Control app
  - Needs to be set to **Broadcast** feed
  - User: **Broadcast25** (if you need to choose it)
- [ ] Switch ATEM panel to M/E 3
- [ ] Check HELO status via web interface
- [ ] Check YouTube and ensure you're on the latest stream event
- [ ] Check Camera 4 position - does it need adjusting?
- [ ] Check comms/talkback to host
- [ ] Update names in ProPresenter for hosts
- [ ] Check any additional assets (e.g., liturgy slides)
- [ ] Rehearse and perform camera check
