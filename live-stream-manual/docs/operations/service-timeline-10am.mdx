# 10am Streamed Service Timeline

This timeline captures the normal flow for streamed 10am services. Times after 10:00 are approximate unless stated.

| Time | Details |
| --- | --- |
| 08:15<br/>T-01:45 | Team arrives, sets up, and checks systems. |
| 09:15<br/>T-00:45 | Team joins pre-service prayer. |
| 09:25<br/>T-00:35 | Director runs the pre-doors comms and readiness checklist. |
| 09:30<br/>T-00:30 | Doors open. IMAG shows pre-service notices from Graphics Producer ProPresenter. |
| 09:49<br/>T-00:11 | Livestream starts with a holding slide from Stream Producer ProPresenter. |
| 09:50<br/>T-00:10 | Livestream hosts start on Camera 6. |
| 09:59:00<br/>T-00:01:00 | Graphics Producer starts 1-minute countdown video with music. |
| 09:59:30<br/>T-00:00:30 | Hosts stop talking. Stream takes final 30 seconds of the countdown. |
| 10:00:00<br/>T+00:00:00 | Countdown ends and service starts. |
| During service<br/>(variable) | For the Peace / children leaving: in-room IMAG shows static parent-instructions slide, while stream goes to hosts on Camera 6. |
| During communion services<br/>(variable) | In-room IMAG shows communion instructions, while stream goes to hosts on Camera 6. |
| ~11:30<br/>T+01:30 | Service aims to finish. Livestream hosts make closing remarks, typically reading/replying to live chat comments. |
| End of stream (immediately after hosts)<br/>T+01:30 to T+01:31 | Graphics Producer hands over to the 12pm IMAG Director. Stream Producer cuts to closing slide and ends stream about 10-20 seconds after hosts finish. |

## Notes

- This is the default 10am pattern. Adapt as needed for unusual service elements.
- Stream health checks start at go-live, not in the 09:25 pre-doors checklist.
