# 8:00 System Startup

This checklist is for whoever arrives first (typically around 8am), usually a Director or Producer. Complete these tasks to initialize the system before other team members arrive.

<Checklist />

## Notes

- **System Initial Setup Button**: This button initializes power relays and brings key systems online. Hold the button as indicated.
- **Allen & Heath Custom Control App Audio**: Verify that the stream audio is correctly routed from the broadcast mix, not the house mix. The [Custom Control app](https://www.allen-heath.com/hardware/controllers/custom-control/) provides remote control of the Allen & Heath mixer routing.
- **HELO Web Interface**: Check encoder status, recording availability, and connection health.
- **YouTube Stream Event**: Ensure you're starting with the correct scheduled stream (not a previous one).
- **Camera 4**: This fixed camera may need repositioning based on the service layout or special requirements.
- **ProPresenter Host Names**: Update lower thirds and graphics with the correct host names for today's service.

## After Completion

Once this checklist is complete, the system should be ready for the rest of the team to arrive and complete their individual role preparations. The [9:25 Pre-Doors Checklist](/operations/pre-doors-checklist) will verify full team readiness before the service begins.
