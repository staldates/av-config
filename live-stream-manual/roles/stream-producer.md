# Stream Producer Role

## Overview

**Location:** Gallery (right seat)

**Primary responsibility:** Managing the livestream and online experience.

**Workstation:** The Stream ProPresenter machine serves as the central control point for all streaming operations, including HELO control, YouTube monitoring, and livestream chat.

The Stream Producer controls the **Stream M/E (M/E 2)** using a Stream Deck and manages the YouTube stream and streaming hardware (HELO). They also monitor the livestream chat and coordinate with hosts when needed.

## Key Responsibilities

- Control the stream output using the Stream Deck
- Start and monitor the YouTube livestream
- Manage the HELO streaming device
- Display slides such as "Starting Soon" and lower thirds using ProPresenter
- Monitor livestream chat and relay information if needed
- Communicate with online hosts (Camera 6)

## Equipment Used

**Stream ProPresenter Machine (Central Workstation):**

- **ProPresenter:** Stream-specific overlays, lower thirds, and graphics
- **HELO Control:** Interface for managing the HELO streaming device
- **YouTube Studio:** Stream monitoring and chat
- **Capture Device:** Takes M/E 2 feed, can be used for OBS, Zoom, or alternative streaming
- **Stream Deck:** Quick access controls for M/E 2 (connected via Bitfocus Companion)
- **Multiple Monitors:** Managing all streaming systems simultaneously

**Additional Equipment:**

- **HELO Streaming Device:** Primary hardware encoder for YouTube output
- **Comms System:** Communication with hosts at Camera 6

**Alternative Streaming:**

- Capture device can be used instead of HELO if needed
- Enables streaming via OBS, Zoom calls, or other platforms
- Also allows for scopes using OBS plugin

## Pre-Service Checklist

- [ ] Power on and check HELO streaming device
- [ ] Launch ProPresenter (Stream machine) and load stream-specific content
- [ ] Verify YouTube Studio is accessible and stream key is configured
- [ ] Test Stream Deck controls for M/E 2
- [ ] Prepare "Starting Soon" slide in ProPresenter
- [ ] Check network connection and stream health
- [ ] Coordinate with online hosts (Camera 6) if in use
- [ ] Open and monitor YouTube livestream chat
- [ ] Verify audio and video quality in YouTube preview

## During Service

### Stream Management

1. **Pre-Service:**
   - Start YouTube livestream 10-15 minutes before service begins
   - Display "Starting Soon" or welcome slide
   - Monitor incoming viewers and chat

2. **Service Start:**
   - Transition from "Starting Soon" to live service feed
   - Ensure M/E 2 is following the service appropriately

3. **During Service:**
   - Monitor stream health (bitrate, dropped frames, connection)
   - Watch for any technical issues and address quickly
   - Add lower thirds or overlays as needed using Stream ProPresenter
   - Coordinate with IMAG Producer so graphics don't conflict

4. **Post-Service:**
   - Keep stream running if there's post-service content
   - End stream when appropriate
   - Archive or process recording as needed

### Stream M/E Control

- Use Stream Deck to control what goes out to the livestream
- May differ from in-room IMAG depending on online needs
- Coordinate with Director and Graphics Producer for consistency

### Chat Monitoring

- Keep an eye on livestream chat for:
  - Technical issues viewers are experiencing
  - Questions that hosts or leadership should address
  - Community engagement
  - Users posting inappropriate content
- Relay important information to relevant team members

### Host Coordination

- Communicate with online hosts at Camera 6
- Coordinate timing for host segments (pre-service, transitions)
- Ensure hosts are cued and ready when needed

## Skills That Help

- Comfort managing several screens at once
- Attention to detail
- Ability to troubleshoot calmly

## Common Issues and Troubleshooting

### Stream Health Issues

- **Dropped frames:** Check network connection, HELO status
- **Audio sync:** Verify HELO audio routing
- **Poor quality:** Check bitrate settings and bandwidth

### YouTube Issues

- **Stream not starting:** Verify stream key, check account status
- **Chat not working:** Check YouTube Studio settings
- **Delay:** Normal 10-30 second delay, inform team if unusual

### HELO Issues

- **Not encoding:** Check power, connections, restart if needed
- **No video:** Verify input from ATEM Stream M/E output
- **No audio:** Check audio routing in HELO settings
- **HELO unavailable:** Use capture device with OBS or alternative streaming software as backup

## Communication

- Coordinate with Director on service flow
- Update Graphics Producer if stream-specific needs arise
- Inform hosts of timing
- Relay viewer feedback if relevant to service team
